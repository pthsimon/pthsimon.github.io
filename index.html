Over the course of your career you may be presented with technical tests or questions from employers. Tests are used by employers to guage whether you are suitable for a job. The following is a set of questions I was given while applying for a web development position. These are also a good way to learn some new technical skills/tricks that you might not have picked in formal education.

I will try to post more here as I apply for additional jobs and perhaps some interview questions later.

What is the use of in_array() function in php?

A1: The in_array() function is used to determine if a value is in an array. There is a simple example at w3shools to illustrate this one. The output would return "Match found" as Glenn is in the array.

2. What is the use of explode() function ?

A2: Tool for manipulating text strings. Eg Separating a string at reference points(Characters or spaces) assigned in the function.

3. Write down the code to save an uploaded file in php.

A3: To save a file to the server from a web page. You will require a location to save the file and a method for the client to choose the file from their system. An actual implementation should include additional file checking and security this code includes some basic file validation.

4. How to remove whitespace from the beginning and end of a string ? use trim() from php’s text processing functions

5. How to redirect a page in php? Place header with the new location at the top of the script and no additional content
